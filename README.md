# Compteur de mot dans un fichier en Java
Ce programme Java permet de compter le nombre de mot dans un fichier et de retourner cette valeur.

* Utlisation de la classe "Scanner" pour parcourir le fichier ligne par ligne
* Utilisation du séparateur " " pour récupérer chaque mot dans chaque ligne
* Utilisation du bloc 'try-catch' pour gérer les erreurs potentielles.
